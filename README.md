# amplitude-builder-skills

**Open-source AI skills for builders — battle-tested by PMs at Amplitude.**

Builders spend too much time on the mechanics of writing specs, synthesizing research, and drafting comms — and not enough time on the thinking behind them. These skills give AI the structure and context to handle the mechanical parts, so you can focus on the decisions that matter.

Each skill is a standalone prompt template designed to work with Claude and other LLMs. Pick the ones that fit your workflow, customize them, and make them yours.

---

## Skills

| Skill | What it does |
|-------|-------------|
| [craft-spec](skills/craft-spec/SKILL.md) | Turn messy ideas into a clear, structured PRD |
| [craft-experiment-design](skills/craft-experiment-design/SKILL.md) | Write a hypothesis, define success metrics, and plan a holdout strategy |
| [craft-experiment-readout](skills/craft-experiment-readout/SKILL.md) | Summarize results, call a winner, and draft a stakeholder-ready recommendation |
| [craft-discovery-synthesis](skills/craft-discovery-synthesis/SKILL.md) | Extract themes and insights from raw user interviews and feedback |

## How to Use

1. Browse the `/skills` folder and find a skill that fits your task.
2. Open the `SKILL.md` file and copy the prompt template.
3. Paste it into your LLM of choice, fill in the placeholders, and run it.
4. Adapt the output to your context — these are starting points, not finished products.

## Contributing

We'd love your help making these skills better. Here's how:

- **Improve an existing skill** — open a PR with changes to any `SKILL.md` file.
- **Add a new skill** — create a new folder under `/skills` with a `SKILL.md` following the existing format.
- **Share feedback** — open an issue describing what worked, what didn't, or what's missing.
